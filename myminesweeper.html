MINESWEEPER

Created by Georgina Halpern and Pamela Needle

Instructions: Left click on a cell to reveal what is underneath, either the number of surrounding mines or a MINE! Right click to flag the cell if you suspect there is a mine underneath. The counter starts off as the number of mines on the grid, and decrements with each cell you flag. The starting number of mines is determined by a set probability for each game level, so the number of mines varies each game. GOOD LUCK!
